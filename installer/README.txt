MGR-S Installer

This directory contains the installer files for MGR-S.

To build the installer:

1. Install Inno Setup 6 (http://www.jrsoftware.org/isdl.php)
2. Run install.bat
3. The installer will be created in the Output directory

To install MGR-S:

1. Run mgrs_setup_0.1.exe from the Output directory
2. Follow the installation instructions

To uninstall MGR-S:

1. Go to Control Panel > Programs and Features
2. Select MGR-S and click Uninstall

To run MGR-S:

1. From the Start Menu, go to MGR-S > MGR-S
2. Or double-click the MGR-S shortcut on the desktop

To test MGR-S:

1. Run the UI from the installation directory
2. Check the log output for any errors

To configure MGR-S:

1. The UI provides a simple interface to view system status
2. No additional configuration is required

To report issues:

1. Check the log output for errors
2. Submit a bug report at https://github.com/mgr-s/mgr-s/issues
